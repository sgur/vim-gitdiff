vim-gitdiff
===========

Description
-----------

A tiny plugin to use `git --histogram` as diffexpr.

Requirement
-----------

* [vimproc](https://github.com/Shougo/vimproc.vim) (*Optional, for Windows)
* git

Usage
-----

```vim
set diffexpr=gitdiff#diffexpr()
```
